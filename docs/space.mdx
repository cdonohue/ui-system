# Space

Use the following utilities to style `margin`, `padding`, `width`, and `height`.

## Margin

<Box
  style={{
    display: "grid",
    gridTemplateColumns: "repeat(auto-fill, minmax(100px, 1fr))",
    gridGap: "1rem",
  }}
>
  <MarginBox caption="All" pattern="m-{size}" marginModifier="m-4" />
  <MarginBox caption="Horizontal" pattern="mx-{size}" marginModifier="mx-4" />
  <MarginBox caption="Vertical" pattern="my-{size}" marginModifier="my-4" />
  <MarginBox caption="Top" pattern="mt-{size}" marginModifier="mt-4" />
  <MarginBox caption="Right" pattern="mr-{size}" marginModifier="mr-4" />
  <MarginBox caption="Bottom" pattern="mb-{size}" marginModifier="mb-4" />
  <MarginBox caption="Left" pattern="ml-{size}" marginModifier="ml-4" />
</Box>

<ConfigPropertyList property="margin" />

> Negative margin utilities are also available. Use `-m{side?}-{size}`
